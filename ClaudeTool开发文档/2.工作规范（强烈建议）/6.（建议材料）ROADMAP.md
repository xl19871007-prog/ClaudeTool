### 阶段 1：MVP（最小可用版本）
- **目标**：能闭环使用的最小功能集，发布给小白用户首测
- **包含功能**（PRD 编号）：F1（含一键安装 Win 原生路径）/ F2 / F4 / F5 / F6 / F7
- **不包含**：F3（暂缓，等官方 API）/ F8（V2）/ npm 路径安装与 Node 自动安装（留 V1.1）/ Skills 一键安装 / macOS / Linux
- **验收标准**：
  - 完全没接触过 Claude Code 的小白，从下载本工具到发出第一句对话 ≤ 5 分钟（含安装 Claude Code）
  - Win10/11 下启动 ≤ **3 秒**，空闲内存 ≤ **150MB**
  - 一键安装成功率 ≥ 90%（在网络通畅的 Win10 1809+ 环境）
  - 命令大全字典 ≥ **30 条**覆盖最高频场景
  - 6 项 P0 功能全部可用，无阻塞性 bug

### 阶段 2：V1.1 跨平台与体验完善
- **目标**：补 macOS / Linux，加 V2 加固功能
- **包含**：一键安装的 npm 路径（含 Node 自动安装）、F8 对话引导（规则版）、命令大全动态化、Skills 一键安装、多代理 profile
- **不包含**：F3（仍等官方 API）、移动端

### 阶段 3：V2.x（按反馈定）
- 候选：F3 Token 监控（如官方放出 API）、对话引导智能版（Haiku Coach）、团队/分享功能、远程 Web 版

## 6. 已搁置 / 已放弃

- ⏸️ **F3 Token 配额监控**（暂缓）：Anthropic 无官方配额查询 API，第三方估算不准会误导小白。等官方公开后启动。详见 ADR-002。
- ⏸️ **F8 对话引导**（推迟到 V2）：MVP 不做，V2 起步走规则引擎。
- ❌ **Electron 方案**：放弃，详见 ADR-001。
- ❌ **内置代理 / VPN 功能**：永久放弃。CLAUDE.md 核心原则 #2。
- ❌ **多账号管理**：MVP 不做，让用户用 `claude auth login` 自己管。
- ❌ **替代 Claude Code 自身的对话**：永久放弃。本工具是包装器，不是替代品。
